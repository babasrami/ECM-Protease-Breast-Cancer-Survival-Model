# Data folder

Place local input data here if preferred.

Required files are not included because they are large, public, or generated locally.

Download METABRIC:

```bash
wget https://datahub.assets.cbioportal.org/brca_metabric.tar.gz
mkdir -p brca_metabric
tar -xzf brca_metabric.tar.gz -C brca_metabric --strip-components=1
```
